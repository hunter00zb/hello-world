- ## 🚀 快速开始

### 前置要求

- Node.js >= 18.0.0
- Git

### 安装

\`\`\`bash
# 克隆仓库
git clone https://github.com/hunter00zb/hello-world.git
cd hello-world

# 安装依赖
npm install

# 启动项目
npm run dev
\`\`\`
