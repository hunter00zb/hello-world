# 如何参与贡献

感谢你有意向为本项目贡献代码！

## 开发环境设置

```bash
# 克隆仓库
git clone https://github.com/hunter00zb/hello-world.git
cd hello-world

# 安装依赖
npm install

# 启动开发模式
npm run dev
```

## 代码规范

- 使用 ESLint 检查代码规范
- 使用 Prettier 格式化代码
- 提交前运行 `npm run lint`

## 提交规范

本项目遵循 [Conventional Commits](https://www.conventionalcommits.org/) 规范：

```
feat: 新增功能
fix: 修复bug
docs: 文档更新
style: 代码格式调整
refactor: 重构
test: 测试相关
chore: 构建/工具相关
```

## Pull Request 流程

1. Fork 本仓库
2. 从 `main` 分支创建新分支
3. 完成开发并测试
4. 提交 Pull Request
5. 等待代码 Review
6. 合并后删除分支

## 发现 Bug？

请通过 [Issue](https://github.com/hunter00zb/hello-world/issues) 反馈，描述清楚问题环境和复现步骤。
